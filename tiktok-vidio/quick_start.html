🎬 抖音视频自动发布工具

快速入门指南 - 从零开始，5分钟上手

1

环境准备

安装 Python

  • 访问 python.org 下载 Python 3.8+
  • 运行安装程序
⚠️ 重要：安装时务必勾选 "Add Python to PATH" 选项！

验证安装

按 Win + R，输入 cmd，回车，然后输入：

python --version

应显示类似：Python 3.10.11

2

一键构建

方法A：最简单（推荐新手） 推荐

1. 打开文件资源管理器

2. 进入 D:\AIproject\tiptok-vidio 目录

3. 双击运行 make_all.bat

4. 等待完成（约2-5分钟）

5. 完成后自动打开 dist 文件夹

✅ 完成后：可执行文件位于 dist\TikTokPublisherWeb.exe

方法B：分步执行 进阶

按 Win + R，输入 cmd，回车，然后依次输入：

# 进入项目目录
cd /d D:\AIproject\tiptok-vidio

# 安装依赖（只需执行一次）
pip install -r requirements.txt

# 运行Web版本
python run_web.py

浏览器会自动打开 http://127.0.0.1:5000

3

生成安装包

简易安装包 推荐

无需安装额外工具，使用Windows自带功能。

步骤：

  1. 双击运行 build_web.bat 打包程序
  2. 找到 installer\install_simple.bat
  3. 右键点击 → 以管理员身份运行
  4. 按照提示完成安装
📦 包含功能：桌面快捷方式、开始菜单、开机自启动、卸载程序

专业安装包 需要Inno Setup

使用Inno Setup创建专业的Windows安装程序。

步骤：

  1. 双击运行 installer\download_inno.ps1 安装Inno Setup
  2. 双击运行 build_installer.bat
  3. 安装包位于 dist\installer\TikTokPublisher_Setup_1.0.0.exe
4

安装部署

在本机安装：

  • 双击运行安装包，按向导完成安装
  • 或以管理员身份运行 install_simple.bat

在其他电脑部署：

  • 方法1：复制 dist 文件夹到目标电脑，双击exe运行
  • 方法2：将安装包复制到目标电脑，双击安装
✅ 安装完成后：桌面会有快捷方式，程序会开机自动启动

常见问题

Q: 运行脚本时提示"python不是内部或外部命令"
原因：Python未添加到系统PATH
解决：重新安装Python，安装时勾选 "Add Python to PATH"
Q: pip install 失败
原因：网络问题
解决：使用国内镜像源：
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
Q: 安装包被杀毒软件拦截
原因：安装包会修改注册表和创建自启动项
解决：临时关闭实时防护，或添加到白名单
Q: 程序无法启动
可能原因：端口5000被占用
解决：重启电脑后重试，或修改端口号
