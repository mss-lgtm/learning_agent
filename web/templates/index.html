抖音账号配置

如何获取 Client Key 和 Client Secret

前置条件：抖音开放平台要求完成企业开发者认证后才能创建网站应用获取凭证。个人开发者无法直接创建网站应用，需要先注册个体工商户或企业资质。
1
注册并登录抖音开放平台

访问 https://open.douyin.com，使用抖音APP扫码登录或手机号注册。

首次登录需要完成开发者资质认证（个人/企业），建议使用企业资质。

抖音开放平台登录页面
2
创建网站应用

登录后，进入左侧「管理中心」→「我的应用」，点击右上角「创建应用」按钮。

  • 应用类型：选择「网站应用」
  • 应用名称：自定义，如"视频发布助手"
  • 应用描述：说明应用用途，如"用于定时自动发布视频到抖音"
提示：如果看不到「创建应用」按钮，请确认已完成企业开发者认证。个人开发者需要先注册个体工商户。
管理中心 - 点击「创建应用」
3
填写应用信息

在创建应用页面，填写以下信息：

  • 应用类型：选择「网站应用」
  • 应用名称：自定义名称
  • 应用描述：说明用途
  • 回调地址：填写 http://localhost:8080/callback
重要：回调地址必须填写 http://localhost:8080/callback，否则授权流程无法完成。
填写应用信息，回调地址填写 http://localhost:8080/callback
4
获取凭证（Client Key 和 Client Secret）

应用创建成功后，进入应用详情页，点击「凭证信息」标签页，可以看到：

  • Client Key：应用的唯一标识（以 aw 开头）
  • Client Secret：应用密钥（点击"查看"按钮显示）

点击右侧「复制」按钮分别复制 Client Key 和 Client Secret。

安全提示：Client Secret 是敏感信息，请勿分享给他人或上传到公开仓库。本工具使用 AES-256 加密存储。
在「凭证信息」页面复制 Client Key 和 Client Secret
5
申请视频发布权限

在应用详情页，点击「权限管理」标签页，申请以下权限：

  • video.create - 创建视频（必选，发布视频必需）
  • video.data - 视频数据（必选，读取视频信息）
  • user.info - 用户信息（可选）
注意：video.create 权限可能需要提交使用场景说明，审核周期约 1-3 个工作日。
在「权限管理」页面申请 video.create 等必要权限
6
在本工具中完成配置

将复制的 Client Key 和 Client Secret 粘贴到下方表单，然后：

  1. 点击「保存凭证」保存配置
  2. 点击「授权登录」打开抖音授权页面
  3. 使用抖音扫码完成授权
  4. 将授权码粘贴到弹出的对话框中
将凭证粘贴到本工具，点击「授权登录」完成配置
常见问题：
  • Q: 个人开发者能否创建应用？
    A: 抖音开放平台要求企业资质。个人可注册个体工商户（免费），再用企业资质认证。
  • Q: 回调地址填错了怎么办？
    A: 在应用详情页 → 安全设置中修改回调地址。
  • Q: Client Secret 忘记了怎么办？
    A: 在凭证信息页面可以重新查看或重置 Client Secret。
